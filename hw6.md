# 介面設計分鏡板

![1](1.png)

| 元件名稱 | 元件功能 |

### 參數設定

![2](2.png)

| 欄位名稱 | 資料型態 | 驗證規則 |
| :--- | :--- | :--- | 
| **關鍵字** |  String | 1. 非必填<br> |

| 元件名稱 | 元件功能 |
| :--- | :--- |
| **關閉按鈕** | 關閉此頁面 |

### 登入系統

![4](4.png)

| 欄位名稱 | 資料型態| 驗證規則 |
| :--- | :--- | :--- | 
| **帳號** |  String | 1. 必填<br> 2. 不可為空 |
| **密碼** |  String  |  1. 必填<br> 2. 不可為空 |

| 元件名稱 | 元件功能 |
| :--- | :--- |
| **關閉按鈕** | 關閉此頁面 |
| **登入按鈕** | 驗證帳密進行登入 |
| **註冊按鈕** | 註冊帳號 |
